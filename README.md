# 사계절 무스펠 신호등 연습기

AION2 성역3 `무스펠의 성배` 2네임드 신호등 기믹을 브라우저에서 연습하는 Canvas 웹앱입니다.

## 특징

- 원본 사이트의 UX 구조를 참고하되 코드/그래픽은 새로 구현
- 1줄/2줄/랜덤 패턴 선택
- WASD/방향키 이동, Shift 대시, Space 시작, R 재시작, P 일시정지
- 줄마다 3개 구슬 세트 표시
- 빨간 줄이 닿은 줄의 3개 구슬이 동시에 Red → Green → Blue 순서로 전환
- 1줄/2줄 패턴과 3·5·7번째 2줄 동시 타격 타이밍 반영
- 구슬을 3D 구체처럼 보이도록 radial gradient, glow, highlight 적용
- 캐릭터도 그림자/몸체/머리/왕관 레이어로 입체감 적용
- 미니맵 목표 표시, 난이도, 카메라 기울기 조절

## 개발

```bash
npm install
npm run start
```

## 배포

GitHub Pages용 워크플로우가 포함되어 있습니다. `main` 브랜치에 push하면 자동 배포됩니다.
